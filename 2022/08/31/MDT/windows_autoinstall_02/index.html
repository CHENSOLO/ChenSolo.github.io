Windows自动化装机部署（二）MDT环境准备

1.下载并安装 Windows ADK

  • 这里我选择默认路径，也可以选其他路径

  • 可根据实际需求选择所要安装的功能

  • 暂时不启动”开始指南”

2.下载并安装MDT2013

  • 暂时不加入，直接选择“Next”

服务器端：AD、DNS、DHCP
客户端：MDT、WDS、ADK 8.1
服务端与客户端所需的环境准备条件到此完成。
在设置准备好MDT环境后，我们需要对MDT、WDS等进行首次使用的简单设置，以方便后续OS、驱动等的导入及分发部署。

3.windows部署服务配置

  • 打开windows部署服务控制台
  • 右键MDT服务器，选择配置服务器

  • 配置界面

  • WDS控制台配置好后会在安装后指定文件下生成相应的文件目录

4.MDT服务台配置

  • 打开 MDT 控制台“Deployment Workbench”并新建

  • 指定网络共享路径，可以自定义设置。

  • 设置共享名称

  • 设置共享描述

  • 相关选项，可根据需要勾选，这里全不选。

  • 配置完成后视图如下，互相对应的

  • 各目录互相对应的信息
